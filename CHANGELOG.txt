FIX #3: CANDIDATE CAN VOTE FOR THEMSELVES
===========================================
Status: ✅ FIXED
File: election.h
Function: vote() - Line 727-743
Change: Added check to prevent candidates from voting in positions they're
        running for

Code Added:
    bool isCandidate = false;
    for (auto& c : list) {
        if (c.roll == voterRoll) {
            isCandidate = true;
            break;
        }
    }

        printWarning("You are a candidate for " + pos + ". Cannot vote!");
        continue;

        election practices.
